HTML

  • An element is a type of content on a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element appears at the bottom of the page and often displays copyright information, contact details and navigation features.
  • Elements like 'head', 'body' and 'footer' are examples of semantic elements because their names describe their content and role in the webpage.
  • The p element formats the text inside the tags into a paragraph.
  • The img element is used to display images. In this element, the src attribute defines the location of the image file, and the alt attribute contains a text description of the image.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border property assigns a border to the element.
  • The box-shadow property adds shadow effects around the element's frame.
  • The text-align property determines where text is positioned in the element.

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: switches to another brance
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes in the current branch to the staging area
  • git commit -m "commit-message": creates of snapshot of the currently staged changes accompanied by the added message
  • git pull origin branch-name: pulls the latest version of the code in the specified branch of the project repo to the local computer
  • git push origin branch-name: pushes the local changes to the specified branch of the project repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • An array is a variable that holds a group of related data.
  • The location of each item in an array is identified with a number called an index.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can be used to interrupt the control flow.
  • Code in a conditional if statement will only execute if the statement meets a condition, or is true.
  • Code in a for loop will repeat a specified number of times.
  • A function is a reusable block of code that performs a specific task.
  • Functions must be defined and then called in order to execute.
